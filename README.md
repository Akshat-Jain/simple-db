# simple-db

# Steps to run

```shell
# 1. Navigate to project root
cd ~/github-personal/

# 2. Configure (generate build files in ./build directory)
cmake -S . -B build

# 3. Build (compile the code using info in ./build directory)
cmake --build build

# 4. Run the executable (it will be inside the build directory)
./build/simple-db
```
